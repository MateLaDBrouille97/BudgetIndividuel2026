1. Objet

La présente politique décrit comment l'add-on Google Sheets "Budget Annuel 2026" traite les données de ses utilisateurs. Nous nous engageons à respecter votre vie privée et à n'utiliser vos données qu'aux fins strictement nécessaires au fonctionnement de l'Add-on.

2. Données collectées

2.1 Données que nous ne collectons PAS

L'Add-on ne collecte, ne stocke et ne transmet AUCUNE des informations suivantes vers nos serveurs :

  • Données financières (dépenses, revenus, soldes, objectifs)
  • Informations personnelles (nom, adresse, numéro de téléphone)
  • Données de localisation
  • Identifiants de connexion Google
  • Contenu de vos Google Sheets

2.2 Données stockées localement dans votre document

Les informations suivantes sont stockées exclusivement dans les propriétés de votre propre document Google Sheets (PropertiesService), accessibles uniquement par vous :

  • Clés API saisies par l'utilisateur (Claude, OpenAI, Gemini) — chiffrées par Google
  • Objectifs financiers créés via l'assistant IA
  • Préférences de configuration (langue, pays, solde de départ)
  • Compteur d'utilisation anonyme (usage local uniquement)

Ces données ne quittent jamais votre document. Elles sont supprimées définitivement si vous désinstallez l'Add-on ou supprimez le document.

2.3 Données transmises aux fournisseurs IA

Lorsque vous utilisez l'assistant IA, les messages sont transmis directement au fournisseur choisi (Anthropic, OpenAI ou Google), avec votre propre clé API, sous votre propre compte :

  • Anthropic (Claude) : anthropic.com/legal/privacy
  • OpenAI (GPT-4o) : openai.com/privacy
  • Google (Gemini) : policies.google.com/privacy

3. Permissions Google OAuth

spreadsheetsLire et écrire dans votre Google Sheets actif (budget, objectifs, paramètres)

4. Partage des données

Nous ne vendons, ne louons et ne partageons aucune donnée utilisateur avec des tiers à des fins commerciales.

5. Sécurité

Les clés API sont stockées via PropertiesService de Google Apps Script, qui chiffre les données au repos. Recommandations :

  • Ne jamais partager votre fichier Google Sheets si des clés API y sont stockées
  • Utiliser des clés API avec des limites de dépenses configurées
  • Régénérer vos clés API en cas de compromission suspectée

6. Conservation et suppression

Supprimez vos données à tout moment via :

  • Extensions → Budget 2026 → Réinitialiser Budget
  • Extensions → Gérer les extensions (désinstallation)
  • Suppression de votre document Google Sheets

7. Enfants

L'Add-on n'est pas destiné aux enfants de moins de 13 ans et ne collecte sciemment aucune donnée les concernant.

8. Modifications

Nous nous réservons le droit de modifier cette politique. En cas de changement substantiel, une notification sera affichée dans l'Add-on lors de la prochaine utilisation.

1. Overview

This Privacy Policy describes how the "Annual Budget 2026" Google Sheets add-on handles user data. We are committed to protecting your privacy and using your data only as strictly necessary to operate the Add-on.

2. Data collected

2.1 Data we do NOT collect

The Add-on does not collect, store, or transmit any of the following to our servers:

  • Financial data (expenses, income, balances, goals)
  • Personal information (name, address, phone number)
  • Location data
  • Google login credentials
  • The content of your Google Sheets

2.2 Data stored locally in your document

The following is stored exclusively in your own Google Sheets document properties (PropertiesService), accessible only by you:

  • API keys entered by the user (Claude, OpenAI, Gemini) — encrypted by Google
  • Financial goals created via the AI assistant
  • Configuration preferences (language, country, starting balance)
  • Anonymous usage counter (local only)

This data never leaves your document. It is permanently deleted if you uninstall the Add-on or delete the document.

2.3 Data transmitted to AI providers

When you use the built-in AI assistant, messages are transmitted directly to your chosen provider (Anthropic, OpenAI, or Google), using your own API key:

  • Anthropic (Claude): anthropic.com/legal/privacy
  • OpenAI (GPT-4o): openai.com/privacy
  • Google (Gemini): policies.google.com/privacy

3. Google OAuth permissions

PermissionUsage
spreadsheetsRead and write to your active Google Sheets file (budget, goals, settings)
script.container.uiDisplay the Add-on's menus, sidebars, and dialogs

script.external_requestCall AI APIs (Anthropic, OpenAI, Google) at your request

4. Data sharing

We do not sell, rent, or share any user data with third parties for commercial purposes.

5. Security

API keys are stored via Google Apps Script's PropertiesService, which encrypts data at rest. We recommend users:

  • Never share their Google Sheets file with unauthorized parties if API keys are stored in it
  • Set spending limits on their API keys in their provider dashboard
  • Regenerate their API keys if a compromise is suspected

6. Retention and deletion

You can delete your data at any time via:

  • Extensions → Budget 2026 → Reset Budget
  • Extensions → Manage extensions (uninstall)
  • Deleting your Google Sheets document

7. Children

The Add-on is not intended for children under 13 and does not knowingly collect any data from them.

8. Changes

We reserve the right to modify this policy. In the event of a material change, a notification will be displayed in the Add-on upon next use.
