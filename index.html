1. Objet

La présente politique de confidentialité décrit comment l'add-on Google Sheets « Budget Annuel » (ci-après « l'Add-on »), développé par ÖTOMATA Agency, traite les données de ses utilisateurs. Nous nous engageons à respecter votre vie privée et à n'utiliser vos données qu'aux fins strictement nécessaires au fonctionnement de l'Add-on.

2. Données collectées

2.1 Données que nous ne collectons pas Non collecté

L'Add-on ne collecte pas, ne stocke pas et ne transmet jamais les catégories de données suivantes vers des serveurs appartenant à ÖTOMATA Agency :

  • Données financières personnelles (dépenses, revenus, soldes, objectifs)
  • Informations personnelles identifiables (nom, adresse postale, numéro de téléphone)
  • Données de localisation
  • Identifiants ou mots de passe Google
  • Contenu de vos fichiers Google Sheets ou autres fichiers Google Drive
  • Emails, contacts, calendriers ou tout autre service Google non mentionné dans cette politique
Important : ÖTOMATA Agency ne dispose d'aucun serveur backend propre. Aucune donnée ne transite par notre infrastructure.

2.2 Données stockées localement dans votre document Local uniquement

Les informations suivantes sont stockées exclusivement dans les propriétés de votre propre document Google Sheets via l'API PropertiesService de Google Apps Script, accessible uniquement par vous :

  • Clés API saisies par vous (Claude / Anthropic, OpenAI, Gemini / Google) — chiffrées au repos par Google
  • Objectifs financiers créés via l'assistant IA
  • Préférences de configuration (langue, pays, solde de départ, mode de stockage)
  • Compteur d'utilisation anonyme et empreinte de document (usage de journalisation locale uniquement)

Ces données ne quittent jamais votre document. Elles sont supprimées définitivement si vous désinstallez l'Add-on ou supprimez le document Google Sheets.

2.3 Données transmises aux fournisseurs d'IA tiers

Lorsque vous utilisez l'assistant IA intégré, les messages que vous saisissez dans la barre latérale sont transmis directement au fournisseur d'IA que vous avez sélectionné, via votre propre clé API personnelle que vous avez obtenue et saisie vous-même. Ces transmissions incluent :

  • Le texte de vos messages
  • Un contexte budgétaire agrégé et anonymisé (totaux mensuels, catégories actives, mois en cours) permettant à l'IA de répondre de façon pertinente

Aucune valeur individuelle nommée, aucune donnée personnellement identifiable ne sont transmises sans votre demande explicite. L'Add-on ne conserve aucun historique de vos échanges avec l'IA. Ces transmissions sont régies exclusivement par les politiques de confidentialité des fournisseurs respectifs :

  • Anthropic (Claude) : anthropic.com/legal/privacy
  • OpenAI (GPT-4o) : openai.com/privacy
  • Google (Gemini) : policies.google.com/privacy

4. Partage des données

Nous ne vendons pas, ne louons pas et ne partageons pas de données utilisateur avec des tiers à des fins commerciales, publicitaires ou analytiques. Les seules transmissions de données sont celles décrites à la section 2.3, effectuées par l'utilisateur lui-même avec sa propre clé API, vers le fournisseur IA qu'il a choisi.

5. Sécurité

Les mesures de sécurité suivantes sont en place :

  • Les clés API sont stockées via PropertiesService de Google Apps Script, qui chiffre les données au repos conformément aux normes Google Cloud
  • Toutes les transmissions vers les APIs IA s'effectuent exclusivement via HTTPS/TLS — aucune communication non chiffrée n'est effectuée
  • L'Add-on n'accède à aucun fichier Google Drive autre que le document Google Sheets dans lequel il est installé
  • L'Add-on n'accède jamais aux emails, contacts, calendriers ou autres services Google de l'utilisateur
  • Le champ urlFetchWhitelist dans appsscript.json limite strictement les appels réseau aux trois domaines API déclarés

Recommandations pour les utilisateurs :

  • Ne jamais partager votre fichier Google Sheets avec des personnes non autorisées si des clés API y sont stockées
  • Configurer des limites de dépenses sur vos clés API dans le tableau de bord de votre fournisseur
  • Régénérer vos clés API en cas de compromission suspectée

6. Conservation et suppression

Toutes les données stockées par l'Add-on résident dans votre document Google Sheets. Vous pouvez les supprimer à tout moment via l'une des méthodes suivantes :

  • Extensions → Budget 2026 → Réinitialiser Budget — supprime toutes les données budgétaires et objectifs
  • Extensions → Gérer les extensions → Désinstaller — supprime les clés API et préférences stockées
  • Suppression directe de votre document Google Sheets — supprime toutes les données de façon irréversible

L'Add-on ne conserve aucune donnée en dehors de votre document.

7. Enfants

L'Add-on n'est pas destiné aux enfants de moins de 13 ans. Nous ne collectons sciemment aucune donnée concernant des enfants de moins de 13 ans. Si vous pensez qu'un enfant de moins de 13 ans a utilisé cet Add-on, veuillez nous contacter à l'adresse indiquée à la section 9.

1. Overview

This Privacy Policy describes how the "Annual Budget" Google Sheets add-on (the "Add-on"), developed by ÖTOMATA Agency, handles user data. We are committed to protecting your privacy and using your data only as strictly necessary to operate the Add-on.

2. Data collected

2.1 Data we do not collect Not collected

The Add-on does not collect, store, or transmit any of the following to servers owned by ÖTOMATA Agency:

  • Personal financial data (expenses, income, balances, goals)
  • Personally identifiable information (name, postal address, phone number)
  • Location data
  • Google login credentials or passwords
  • Content of your Google Sheets files or other Google Drive files
  • Emails, contacts, calendars, or any other Google service not mentioned in this policy
Important: ÖTOMATA Agency operates no backend servers. No data passes through our infrastructure.

2.2 Data stored locally in your document Local only

The following information is stored exclusively in your own Google Sheets document properties via Google Apps Script's PropertiesService API, accessible only by you:

  • API keys entered by you (Claude / Anthropic, OpenAI, Gemini / Google) — encrypted at rest by Google
  • Financial goals created via the AI assistant
  • Configuration preferences (language, country, starting balance, storage mode)
  • Anonymous usage counter and document fingerprint (local logging purposes only)

This data never leaves your document. It is permanently deleted if you uninstall the Add-on or delete the Google Sheets document.

2.3 Data transmitted to third-party AI providers

When you use the built-in AI assistant, the messages you type in the sidebar are transmitted directly to the AI provider you selected, using your own personal API key that you obtained and entered yourself. These transmissions include:

  • The text of your messages
  • Aggregated and anonymized budget context (monthly totals, active categories, current month) to enable relevant AI responses

No individually named value, no personally identifiable data is transmitted without your explicit request. The Add-on retains no history of your AI conversations. These transmissions are governed exclusively by the respective privacy policies of the AI providers:

  • Anthropic (Claude): anthropic.com/legal/privacy
  • OpenAI (GPT-4o): openai.com/privacy
  • Google (Gemini): policies.google.com/privacy

3. Google OAuth permissions

4. Data sharing

We do not sell, rent, or share user data with third parties for commercial, advertising, or analytics purposes. The only data transmissions are those described in section 2.3, carried out by the user themselves using their own API key, to the AI provider they selected.

5. Security

The following security measures are in place:

  • API keys are stored via Google Apps Script's PropertiesService, which encrypts data at rest per Google Cloud standards
  • All transmissions to AI APIs are made exclusively over HTTPS/TLS — no unencrypted communications are performed
  • The Add-on accesses no Google Drive files other than the active Google Sheets document it is installed in
  • The Add-on never accesses the user's emails, contacts, calendars, or other Google services
  • The urlFetchWhitelist field in appsscript.json strictly limits outbound network calls to the three declared API domains

We recommend users:

  • Never share their Google Sheets file with unauthorized parties if API keys are stored in it
  • Set spending limits on their API keys in their provider's dashboard
  • Regenerate their API keys if a compromise is suspected

6. Retention and deletion

All data stored by the Add-on resides in your Google Sheets document. You can delete it at any time via:

  • Extensions → Budget 2026 → Reset Budget — removes all budget data and goals
  • Extensions → Manage extensions → Uninstall — removes API keys and stored preferences
  • Deleting your Google Sheets document — permanently and irreversibly removes all data

The Add-on retains no data outside your document.
